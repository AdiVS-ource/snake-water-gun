# Snake-Water-Gun Game 🐍💧🔫

A simple Python terminal game based on the classic **Snake-Water-Gun** logic (like Rock-Paper-Scissors).

## 🎮 Rules
- **Snake drinks Water → Snake wins**
- **Water damages Gun → Water wins**
- **Gun shoots Snake → Gun wins**
